Fish Data Policy

No unverified data as fact

No unverified real fish data will be published as fact on this site. All entries will be clearly marked with their verification status.

Manual verification required

Fish locations, time windows and weather notes should be manually checked before inclusion. Automated scraping will not be used without human review.

Placeholder rows are layout-only

Placeholder rows are only used to test layout and browsing functionality. They contain no real game data and should not be treated as accurate.

Separating confirmed data from tips

Future updates may separate officially confirmed data from community-sourced tips, with clear labels for each category.

Frequently Asked Questions

Are these real Heartopia fish?

No. All fish rows currently on this page are sample placeholders used for layout testing. They do not represent real Heartopia fish data.

When will verified fish data be added?

Verified fish data will be added once each entry has been manually confirmed through in-game testing or reliable community sources. There is no set release date for the first verified batch.

Can I search fish on this page?

Yes. A basic search box is available above the fish table. It filters rows by fish name, location or any visible text. This is a local browser-side filter only.

Will this page include time and weather filters?

Time and weather filtering is planned for a future update. The current MVP includes a simple text search. Category-based filters will be added once the database contains real data.

Is this an official Heartopia fish database?

No. This is an unofficial fan-made guide page. It is not affiliated with XD Games or the Heartopia development team.
