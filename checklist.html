Planning Checklist

A set of planning questions to help you organize your learning and reference notes.

What do I want to learn today?

Choose one topic to focus on. This could be a game mechanic, a guide page, or a general discovery question.

Which guide page should I check next?

Rotate through different guides to gradually build a complete understanding of the game.

Did I mark uncertain information as unverified?

Keep track of what is still unconfirmed. Avoid treating placeholder data as verified facts.

Did I avoid treating placeholder data as confirmed?

Placeholder data exists for layout testing only. Do not rely on it for gameplay decisions.

Should I return later after updates?

The guides will improve as more data is verified. Bookmark useful pages and check back periodically.

Verified vs General Tips

Understanding what is safe to rely on and what still needs verification.

General tips

Safe planning advice that does not depend on specific game facts. These can be used freely as reference.

Verified game tasks

Specific game facts that have been manually checked. These will be clearly labeled when added.

Placeholder data

Layout or schema testing only. Contains no confirmed game information. Do not treat as real data.

Unverified tips

Community advice or unconfirmed discoveries. Should not be treated as confirmed until manually checked.

Confirmed data

Will be added only after manual review. All confirmed data will include a source or verification note.
