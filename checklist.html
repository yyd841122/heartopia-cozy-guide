Frequently Asked Questions

Does this checklist save progress?

Not yet. The current MVP version does not save progress. Checkboxes are visual controls only. Future versions may add browser-based or account-based saving.

Are these official Heartopia daily tasks?

No. The checklist items shown are general planning suggestions for MVP layout testing. They do not represent official Heartopia daily tasks.

Will this page include reset timers?

Reset timers are planned for a future update. The current MVP shows static checkbox groups only without time-based features.

Can I use this on mobile?

Yes. The page uses the same responsive layout as the rest of the guide site and works on mobile screens.

Is this an official Heartopia tool?

No. This is an unofficial fan-made guide page. It is not affiliated with XD Games or the Heartopia development team.
