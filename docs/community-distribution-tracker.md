# Cozy Guide Hub 社区分发记录表

## 用途说明

本记录表用于跟踪 Cozy Guide Hub 在不同社区和内容平台上的分发动作，包括用户真实问题、回复内容、是否使用链接、使用哪个页面链接、收到什么反馈、是否带来点击或后续优化线索。

1. 不伪造反馈
2. 不刷屏
3. 不硬广
4. 不伪装成普通玩家推荐
5. 只在链接真正有帮助时使用
6. 优先回答问题，再考虑附链接

| Platform | Distribution Approach |
| --- | --- |
| Reddit | Answer specific questions, avoid hard advertising. Best for "where to start" type posts. |
| Discord | Join discussions naturally, do not spam links. |
| Steam Community | Reply to guide requests, link only when relevant. |

| Personal social / groups | Share with friends who may find it useful. |

## 分发原则

- [ ] 先回答用户问题，再考虑放链接
- [ ] 不刷屏
- [ ] 不伪装成第三方用户推荐
- [ ] 不夸大网站能力
- [ ] 不声称本站是官方
- [ ] 不发布假兑换码
- [ ] 不把未验证数据说成 confirmed
- [ ] 不复制竞品文案
- [ ] 不使用官方图片
- [ ] 每次分发后记录平台和链接
- [ ] 每次分发后记录反馈
- [ ] 没有反馈也要记录为 no visible feedback
