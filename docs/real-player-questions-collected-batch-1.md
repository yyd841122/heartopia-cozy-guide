### 搜索缺口说明

Only 10 real public Chinese questions/topics were found in this pass. English searches (Google, Reddit, YouTube, DuckDuckGo) all returned transport errors. The English game community for Heartopia was not accessible.

Reasons for gap:

1. **English platforms blocked**: Google, Reddit, YouTube, DuckDuckGo all returned transport errors. Cannot confirm English player questions.
2. **Steam no match**: The Steam app ID returned an unrelated game — Heartopia may not be on Steam, or the ID was incorrect.
3. **TapTap search empty**: TapTap search for 心动小镇 returned no direct results — the app page likely exists but search indexing is limited.
4. **知乎 blocked**: Returned 403.
5. **Only Baidu accessible**: Baidu was the only search engine that returned usable results, all in Chinese.
6. **Questions vs guides**: Most Baidu results are guide articles, not direct player questions. Only ~4-5 are actual forum questions; the rest are implied search intent.

**To be completed**: English language player questions require manual Reddit/YouTube/Discord browsing, which was not possible through automated public web search in this pass.
