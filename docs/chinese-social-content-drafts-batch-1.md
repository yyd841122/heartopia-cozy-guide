# 中文社媒图文草稿 Batch 1

## 使用说明

本文件用于准备第一批中文社媒图文草稿，可用于小红书、B站动态、抖音图文、朋友圈或中文玩家社群的人工发布参考。

1. 草稿不能自动群发。
2. 发布前必须根据平台和语境修改。
3. 不硬广。
4. 不刷屏。
5. 不伪造用户反馈。
6. 不伪造点击数据。
7. 不冒充第三方推荐自己网站。
8. 不发布假兑换码。
9. 不把未验证信息写成 confirmed。
10. 明确 Cozy Guide Hub 是 fan-made / 非官方 MVP。

## 可使用页面

| 页面 | URL | 适合场景 |
| --- | --- | --- |
| 首页 | https://cozyguidehub.com/ | 简单介绍网站，不作为默认主推链接 |

## 小红书图文草稿

### 草稿 1：新手入门

**标题：**
Heartopia 新手刚开始玩，先别急着冲进度

**正文：**
刚开始玩 Heartopia，很容易一上来就到处点、到处找攻略。
但新手阶段我更建议先做几件事：

1. 先慢慢看早期提示
2. 不要急着相信各种未验证攻略
3. 兑换码不要乱试假码
4. 先建立一个简单的新手路线
5. 把不确定的信息先记录下来，等验证后再用

我整理了一个 fan-made 新手攻略页，会把未验证内容和已确认内容分开：
https://cozyguidehub.com/beginner-guide.html

**说明：**
这个站不是官方站，目前还是 MVP，会慢慢补充真实验证内容。

### 草稿 2：首日规划

**标题：**
Heartopia 第一天怎么玩？先用一个轻量路线规划

**正文：**
如果你刚开始玩 Heartopia，不一定要马上追求最优路线。
第一天更重要的是：

1. 先熟悉基础操作
2. 看懂早期提示
3. 不急着乱花资源
4. 先建立一个简单日常节奏
5. 遇到不确定攻略先别当真

我整理了一个非官方 First Day Guide：
https://cozyguidehub.com/first-day-guide.html

它主要是通用新手规划，不会把未验证数据写成事实。

### 草稿 3：兑换码防坑

**标题：**
Heartopia 兑换码别乱信，先看有没有 verified

**正文：**
很多游戏刚火的时候，网上最容易出现一堆"可用兑换码"。
但问题是：很多码可能没验证、已过期，甚至是假的。

所以我做了一个 Codes Tracker：
https://cozyguidehub.com/codes.html

目前原则很简单：

1. 没有验证过的码，不放到 active 区域
2. 没有 verified active codes，就明确写没有
3. 不为了流量乱写假兑换码
4. 后续只记录经过检查的兑换码状态

这是 fan-made 页面，不是官方兑换码页。

### 草稿 4：每日清单

**标题：**
Heartopia 每天上线不知道干什么？先用轻量 checklist

**正文：**
如果你每天上线不知道先做什么，可以先用一个简单清单帮自己整理节奏。

我现在整理的是一个 fan-made Daily Checklist：
https://cozyguidehub.com/checklist.html

注意：
它不是官方每日任务表。
现在主要是通用规划建议，比如：

1. 看看当前目标
2. 回顾新手攻略
3. 不乱用未验证兑换码
4. 记录还没确认的信息
5. 后续再根据真实验证数据更新

## 不要发布的中文内容类型

1. 不要说"全网最全"
2. 不要说"官方攻略"
3. 不要说"最新可用兑换码大全"
4. 不要写假兑换码
5. 不要承诺一定有 active codes
6. 不要把 placeholder 数据当真实数据
7. 不要复制竞品文案
8. 不要用官方图片
9. 不要冒充路人推荐
10. 不要刷屏
11. 不要为了吸引点击夸大网站能力
12. 不要说已经有完整数据库

## 记录要求

每次使用中文草稿后，都要记录到 `docs/community-distribution-tracker.md`，至少记录：

1. Date
2. Platform
3. Community / Channel
4. User Question / Topic
5. Response Summary
6. Linked Page
7. Link Used
8. Feedback
9. Click / Visible Result
10. Next Action
