# 真实玩家问题寻找方法与问题池模板

## 为什么当前不能直接选 3 个问题

当前 cozyguidehub.com 刚上线，尚未开始真实社区分发，也没有用户提供真实社区问题。因此不能编造 Reddit / Discord / 小红书 / B站 / 抖音玩家问题。

正确做法是：

1. 先建立问题寻找方法
2. 再人工去平台寻找真实问题
3. 记录至少 10 个真实问题
4. 再从中选择 3 个低风险问题
5. 最后准备回复草稿

## 问题分类规则

1. Beginner question -> beginner-guide.html
2. First day / first session question -> first-day-guide.html
3. Code / reward / redeem question -> codes.html
4. Routine / checklist / daily planning question -> checklist.html
5. Recipe question -> recipes.html，但当前数据仍需验证
6. Fish question -> fish.html，但当前数据仍需验证
7. NPC gift question -> npc-gifts.html，但当前数据仍需验证
8. General resource question -> cozyguidehub.com 首页或更具体页面

优先选择 Beginner / First Day / Codes / Checklist 相关问题，因为这些 P0 页面已经完成线上验证。

## 风险等级定义

**Low risk：**
问题是泛新手建议、首日规划、每日规划，不需要具体游戏事实。

**Medium risk：**
问题涉及 codes、具体系统、具体数据，但可以通过说明 verification boundary 安全回答。

**High risk：**
问题要求具体 recipe、fish location、NPC gift、active code、exact reward、exact route，而当前没有 verified data。高风险问题不应直接回答具体内容。

## 找不到问题时怎么办

如果暂时找不到真实玩家问题，不要编造。可以先做：

1. 扩大搜索关键词
2. 换平台搜索
3. 搜英文关键词
4. 搜中文关键词
5. 看 YouTube 视频评论区
6. 看 Reddit 新帖和旧帖
7. 看小红书 / B站 / 抖音评论
8. 先记录"没有找到"的平台和关键词
9. 根据找不到的问题调整页面方向
