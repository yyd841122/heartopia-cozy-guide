# Cozy Guide Hub 第一批流量入口页面计划

## 1. 当前判断

Cozy Guide Hub 已经完成基础站点、正式域名和 SEO Phase 1，但目前还缺少真正能带来搜索流量和社区转发的内容资产。

现在不是继续装修网站，而是要开始建设"玩家会搜索、会点击、会收藏、会分享"的入口页面。

## 2. 流量来源拆解

**Search traffic**

来自 Google / Bing 等搜索。

用户可能搜索：

- heartopia beginner guide
- heartopia first day guide
- heartopia codes
- heartopia recipes
- heartopia fish
- heartopia npc gifts
- heartopia daily checklist
- heartopia download
- heartopia wiki
- heartopia tips

**Direct traffic**

来自用户收藏、直接输入域名、浏览器历史、社区反复点击。

**Social traffic**

来自 Reddit、Discord、TikTok、YouTube Shorts、小红书、B站、抖音等。

**Referral traffic**

来自其他站点、博客、论坛、社区帖子引用。

**Returning users**

来自工具型页面，比如 checklist、codes tracker、fish tracker、npc gifts tracker。

## 5. 第一批推荐入口页面

### A. 立即优先做的 P0 页面

- Heartopia Beginner Guide
- Heartopia First Day Guide
- Heartopia Codes Tracker
- Heartopia Daily Checklist

**为什么优先**：

- 更容易人工验证
- 不依赖完整数据库
- 适合 SEO
- 适合社区分发
- 适合引导用户收藏

### B. 第二优先级 P1 页面

- Heartopia Recipes Guide
- Heartopia Fish Guide
- Heartopia NPC Gifts Guide
- Heartopia Download / Platforms Guide

这些关键词有搜索需求，但 Recipes / Fish / NPC Gifts 必须等待真实数据验证流程成熟后逐步发布。Download / Platforms 页面可以作为信息型入口，但要避免写未经确认的平台信息。

### C. 暂缓 P2 页面

- Interactive Map
- House Design Ideas
- Events Tracker
- Materials Database
- Full multilingual pages

**为什么暂缓**：

- 工作量大
- 数据验证成本高
- 容易维护困难
- 现在还不是最小闭环

## 6. P0 页面内容结构建议

### Heartopia Beginner Guide

- Who this guide is for
- First 30 minutes
- First day route
- What to unlock first
- Common beginner mistakes
- What is still unverified
- Internal links

### Heartopia First Day Guide

- First login checklist

### Heartopia Codes Tracker

- Current verified codes
- No unverified codes policy
- How to redeem

### Heartopia Daily Checklist

- General daily routine
- Verified vs general tips
- Beginner checklist
- Planning checklist
- Future save feature

## 7. 社区分发计划

**平台建议**：

- Reddit
- Discord
- Steam Community
- YouTube comments
- 小红书
- B站
- 抖音

**规则**：

- 不要硬广
- 先回答问题
- 只有在链接有帮助时才附上
- 不要刷屏
- 不要伪装用户
- 不要夸大数据
- 不要发假兑换码

**每日动作**：

- 每天找 3 个玩家问题
- 认真回答
- 记录问题和链接
- 如果适合，再附 cozyguidehub.com 相关页面
