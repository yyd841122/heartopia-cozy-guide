# 第一条社区回复人工执行包

## 用途说明

本文件用于帮助用户人工执行第一条社区回复。它只提供发布前检查、可复制回复草稿、无链接版本、带链接版本、发布后记录模板。

1. 本文件不是发布记录。
2. 本文件不代表已经回复玩家。
3. 用户必须人工确认平台规则后再发布。
4. 如果平台不适合放链接，优先使用无链接版本。
5. 如果回复看起来像广告，不要发布。
6. 发布后必须记录到 community distribution tracker。

## 发布前检查

- [ ] 我已经重新查看原始问题
- [ ] 原问题仍然适合回复
- [ ] 回复先回答问题，不是只丢链接
- [ ] 链接和问题直接相关
- [ ] 平台允许或至少不禁止外链
- [ ] 回复不像硬广
- [ ] 没有具体未验证游戏事实
- [ ] 没有具体兑换码
- [ ] 没有假兑换码
- [ ] 没有声称本站是官方
- [ ] 没有说"全网最全""官方攻略""完整数据库"
- [ ] 没有复制竞品文案
- [ ] 没有冒充第三方用户
- [ ] 同一社区没有短时间重复发类似链接
- [ ] 发布后准备记录到 tracker

## 无链接回复版本

如果刚开始玩，第一天我建议先别急着追求最优路线，可以先熟悉基础操作、看懂早期提示、别急着乱花资源，然后给自己建立一个简单节奏。

前期最重要的是先知道自己每天上线要看什么、记录什么、哪些信息还没确认。网上看到的攻略、兑换码、路线建议，最好先判断是不是已经验证过，不要直接当成事实。

## 发布后 24 小时观察事项

1. 是否有回复
2. 是否有点赞
3. 是否被删除
4. 是否被认为是广告
5. 是否有人点击或提到网站
6. 是否有人提出新的问题
7. 是否暴露页面内容缺口
8. 是否适合继续发第二条

## 停止条件

如果出现以下情况，停止继续发：

1. 回复被删除
2. 被认为是广告
3. 平台不允许链接
4. 用户反感链接
5. 同社区已经发过类似链接
6. 需要具体未验证游戏事实
7. 需要具体兑换码
8. 账号出现风险提示
9. 你无法客观记录反馈
