# Batch 1 低风险玩家问题筛选记录

## 筛选目的

本文件用于从公开真实玩家问题 Batch 1 中筛选 3 个低风险问题，作为后续回复准备对象。

筛选目标不是追求最大曝光，而是优先选择：

1. 问题真实
2. 来源明确
3. 风险较低
4. 可以先用通用建议回答
5. 不需要编造具体游戏事实
6. 不需要提供真实或假兑换码
7. 可以自然匹配 Cozy Guide Hub 的 P0 页面
8. 回复后可以记录反馈

## 筛选原则

- [x] 问题来自 Batch 1 已记录的真实公开来源
- [x] 问题不要求具体未验证游戏数据
- [x] 问题不要求提供具体兑换码
- [x] 问题不要求具体 NPC 礼物
- [x] 问题不要求具体鱼类位置
- [x] 问题不要求具体食谱材料
- [x] 问题可以通过 general advice 安全回答
- [x] 问题可以匹配 Beginner / First Day / Codes / Checklist 页面
- [x] 链接可以作为补充，而不是硬广
- [x] 回复后可以记录到 community distribution tracker
