Frequently Asked Questions

Is this an official Heartopia website?

No. This is an unofficial fan-made guide site. We are not affiliated with XD Games or the Heartopia team.

Are the codes real?

Not yet. The codes shown on this MVP page are sample placeholders for layout testing only. Real working codes will be added to the Codes section once they have been manually verified. Do not attempt to use the placeholder strings in the game.

Will this site include a full database?

That is the goal. We plan to expand each database section with community-sourced, manually verified data for recipes, fish, NPC gifts and more. The current tables contain sample data to demonstrate the layout and are not accurate game information.

Can I use this on mobile?

Absolutely. The layout is designed to work on both desktop and mobile devices.

What will be added next?

Upcoming additions include expanded database tables, search and filter features, a full NPC gift tracker, and printable checklists.
