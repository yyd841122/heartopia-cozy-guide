Recipe Data Policy

No unverified data as fact

No unverified real recipe data will be published as fact on this site. All entries will be clearly marked with their verification status.

Manual verification required

Data should be manually checked before release. Automated scraping or bulk imports will not be used without human review.

Placeholder rows are layout-only

Placeholder rows are only used to test layout and browsing functionality. They contain no real game data and should not be treated as accurate.

Community-sourced with attribution

Future updates may cite official or community-confirmed sources where appropriate. Sources will be noted alongside each recipe entry.

Frequently Asked Questions

Are these real Heartopia recipes?

No. All recipe rows currently on this page are sample placeholders used for layout testing. They do not represent real Heartopia game recipes and should not be used as game reference.

When will verified recipes be added?

Verified recipe data will be added once each entry has been manually confirmed through in-game testing or reliable community sources. There is no set release date for the first verified batch.

Can I search recipes on this page?

Yes. A basic search box is available above the recipe table. It filters rows by recipe name, ingredients, or any visible text. This is a local browser-side filter only.

Will this page include ingredient filters?

Ingredient-based filtering is planned for a future update. The current MVP includes a simple text search. Category and ingredient tag filters will be added once the database contains real data.

Is this an official Heartopia database?

No. This is an unofficial fan-made guide page. It is not affiliated with XD Games or the Heartopia development team.
