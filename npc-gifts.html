NPC Gift Data Policy

No unverified data as fact

No unverified real NPC gift data will be published as fact on this site. All entries will be clearly marked with their verification status.

Manual verification required

NPC names, gift preferences and friendship notes should be manually checked before inclusion. Automated scraping will not be used without human review.

Placeholder rows are layout-only

Placeholder rows are only used to test layout and browsing functionality. They contain no real game data and should not be treated as accurate.

Separating confirmed data from tips

Future updates may separate officially confirmed data from community-sourced tips, with clear labels for each category.

Frequently Asked Questions

Are these real Heartopia NPC gifts?

No. All NPC gift rows currently on this page are sample placeholders used for layout testing. They do not represent real Heartopia NPC or gift data.

When will verified NPC gift data be added?

Verified NPC gift data will be added once each entry has been manually confirmed through in-game testing or reliable community sources. There is no set release date for the first verified batch.

Can I search NPC gifts on this page?

Yes. A basic search box is available above the NPC gift table. It filters rows by NPC name, gift name or any visible text. This is a local browser-side filter only.

Will this page include friendship level notes?

Friendship level notes are planned for a future update. The current MVP includes a simple text search and placeholder data. Detailed tier information will be added once the database contains real data.

Is this an official Heartopia NPC database?

No. This is an unofficial fan-made guide page. It is not affiliated with XD Games or the Heartopia development team.
